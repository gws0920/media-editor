# VideoEditer

## 介绍
视频剪辑器

## 软件架构
- `vite2`
- `react`
- `typescript`
- Icon: [Google Fonts](https://fonts.google.com/icons)
- 假数据: `mockjs`
- canvas交互: `konva`, `react-konva`


## 安装教程

```
<!-- 安装依赖 -->
yarn

<!-- 运行 -->
yarn dev

<!-- 启动mock数据 -->
yarn mock
```
